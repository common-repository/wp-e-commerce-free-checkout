== Description ==

Allows for Free Checkout with WordPress e-Commerce Plugin.
Can be used with 100% discount coupons and $0.00 products.
When the cart total is $0.00 the payment form will be replaced with the text 'Free Checkout' and offsite checkout will be skipped.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You can use 100% off coupons and $0.00 product prices.
